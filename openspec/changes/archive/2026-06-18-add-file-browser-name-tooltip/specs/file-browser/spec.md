## ADDED Requirements

### Requirement: フルネームのホバーツールチップ

ファイルブラウザの各タイルは、カラム幅により名前が省略表示される場合があるため、ホバー時に省略されていないフルネームをツールチップで表示しなければならない（SHALL）。表示には Flutter 標準の `Tooltip` を用い、名前が省略されているか否かに関わらずホバー時は常に表示する。

- フォルダタイルのツールチップ内容は、そのフォルダの表示名（`displayName`）とする。
- ファイルタイルのツールチップ内容は、そのファイル名（`name`）とする。

ツールチップの追加は、既存のタイルのレイアウト・選択時の視覚的強調・タップ挙動を変更してはならない（SHALL NOT）。

#### Scenario: フォルダタイルにホバーすると表示名のフルネームが出る

- **WHEN** ユーザーがフォルダタイルにマウスをホバーする
- **THEN** そのフォルダの `displayName` のフルネームがツールチップで表示される

#### Scenario: ファイルタイルにホバーするとファイル名のフルネームが出る

- **WHEN** ユーザーがファイルタイルにマウスをホバーする
- **THEN** そのファイルの `name` のフルネームがツールチップで表示される

#### Scenario: 選択中のファイルタイルでもツールチップが表示される

- **WHEN** あるファイルが選択され装飾が適用された状態で、ユーザーがそのタイルにホバーする
- **THEN** ファイル名のフルネームがツールチップで表示され、選択時の背景・先頭アクセントバー・太字タイトルの装飾は維持される
